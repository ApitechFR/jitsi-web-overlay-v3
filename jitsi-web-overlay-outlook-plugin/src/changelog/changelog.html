📦 Changelog – Joona Meet

🆕 Version 1.1.0 – Juillet 2025

  • Nouveau : Ajout d'une logique flexible pour la génération des noms de réunions :
    • Variable ROOM_NAME_PREFIX pour choisir le format souhaité :
      • Nom structuré (alea_name) – génère un nom lisible, par exemple : ChapelleVictorHugoAnalyser-5HJTXDLuHD
      • Préfixe fixe – utilise un préfixe personnalisé, par exemple : Salle-A8X9Z12Yq pour ROOM_NAME_PREFIX="Salle" et ROOM_NAME_LENGTH=15
      • ID brut – génère un identifiant 100 % alphanumérique, sans préfixe, par exemple : R8K2Q4Z1X9W3 pour ROOM_NAME_PREFIX="" et ROOM_NAME_LENGTH=12
    • Variable ROOM_NAME_LENGTH pour définir la longueur totale du nom généré (préfixe + séparateur + suffixe alphanumérique).

🛠️ Version 1.0.4 – Juin 2025

  • Retrait temporaire de la fonctionnalité d'affichage automatique du numéro Voxify et du code PIN En raison d'incompatibilités techniques liées à Axios sur certaines plateformes, cette fonctionnalité a été désactivée afin d'assurer une stabilité optimale de l'application. Elle sera réintégrée une fois le problème résolu.

🛠️ Version 1.0.3 – Avril 2025
